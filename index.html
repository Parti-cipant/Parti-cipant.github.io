Daily Summary

Move the source code of the hexo blog to my own laptop

As the blog is created on the company computer first, I need to copy the source code to my own laptop so that I can write something on my own computer. Therefore, after searching the Internet, I made a new branch and modifed the .gitignore . Then I cloned the branch to my laptop. After installing some required packages for hexo, it worked very well. I thought there would be some problems but luckily the transfering process was very smooth.

Hexo failed to load css stylesheets

Today, after modifying the _config.yml, I suddenly found that the style for my blog disappeared but it remains on my local website. When I was surfing the Internet for a solution, I found that almost all the solutions were too old. After checking the code for introducing the stylesheet, I found that the path to the css file was wrong. Then after google it again, I changed the #URL part in _config.yml. Then the problem was solved.

Slicing for dataframe structure

The slicing and comparison for the dataframe cost mne a whole morning. Try not to use multiple brackets to locate or slice a dataframe. Instead, using .loc() and .at() is always a better choice. Try not to use brackets until the dataframe is reduced to 1-d.

Introduction to this website

Why creating this new blog

As I started an 6-month placement from July, there are a lot of new things for me to learn. Therefore, I plan to have a record of the problems and difficulty that I have met with to offer a reference to the future. Also, I going to write down some personal feeling and interesting experiences. What’s more, my written English has been backsliding since I entered University, so this blog is used for practicing my written English as well. As I don’t want to write those messy html and css files, I still chose hexo as the frame for this website.

Why this theme

When I was in my high school, during COVD-19, I have already started working on my own blog through GitHub Pages. When I looked back to that time, some complicated emotion just came to me, including the miss for that interesting period, the shock of the changes happened on myself, etc. This website is still under the frame Hexo. It used to have a theme called sakura, but for me now, it is too fancy. Now, I prefer things that are concise and informative. Maybe this is a the signal that I am getting older.
